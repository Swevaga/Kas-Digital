Panduan Setup

Google Drive Setup:

  1. Buka Google Cloud Console
  2. Buat project baru
  3. Aktifkan Google Drive API
  4. Buat OAuth 2.0 Client ID
  5. Tambahkan origin URL ke Authorized JavaScript origins

GitHub Setup:

  1. Push kode ke repository GitHub
  2. Aktifkan GitHub Pages di Settings
  3. GitHub Actions akan otomatis deploy
